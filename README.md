# Student Performance Analytics ETL Pipeline

## Project Overview

This project demonstrates an end-to-end ETL (Extract, Transform, Load) pipeline using Python, Pandas, SQLite, and SQL.

## Workflow

1. Extract student data from CSV
2. Transform and clean data using Pandas
3. Generate grades based on marks
4. Save processed data
5. Load data into SQLite database
6. Perform SQL analytics

## Technologies Used

- Python
- Pandas
- SQLite
- SQL

## Key Insights

- Course-wise average marks
- Subject-wise average marks
- Grade distribution
- Highest attendance student
- Top performing students

## Files

- student_data_100.csv → Raw dataset
- student_analysis.py → Data transformation and analytics
- processed_student_data.csv → Processed dataset
- student_sql.py → SQL queries and reporting
